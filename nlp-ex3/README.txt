1.ex3.py
This is the Python file containing the implementation of the Bigram Hidden Markov Model (HMM).
Key features of the script:
- Calculates transition and emission probabilities.
- Implements the Viterbi algorithm for sequence prediction.
- Handles add-one smoothing and pseudo-word transformations.
- Computes error rates for known and unknown words.


2.NLP-EX3.pdf
This PDF contains the solutions to the exercise questions and analysis.
Breakdown of content:
Question 1: Step-by-step solution for transition and emission probabilities and applying the
            Viterbi algorithm to a given sequence.
Question 2: Solution and explanation for the second question in the exercise , which involves
            designing and completing a pseudo-code for a four-gram Viterbi algorithm.
Question 3: This section presents the results obtained from running the Python script (ex3.py)
            and provides an analysis of those results.
            - Comparisons of error rates (known, unknown, and total) for various configurations:
            baseline, HMM without smoothing, HMM with add-one smoothing, and pseudo-words.
            - Insights and discussions comparing the results across methods and configurations.
